Historical Working Relationship

  • When the team was smaller, requests landed informally and the team handled them — it worked because the volume was manageable
  • Before the formal hub-and-spoke model, operational and executional tasks were handled by marketing operations, which sat within marketing. Although helpful at the time, this approach often led to unreliable data, more back and forth, and frustration within the team. This was led by an eager and intelligent analyst, who did not have the formal training on how to build things to scale and last.
  • Today is about catching up — putting a clear structure in place so both teams know what to expect

How Requests Work

The test: Does this require cross-system thinking, data modelling, or technical expertise the marketing team doesn't have? If yes → Tier 1.
1
Ops Analysts
Complex builds, infrastructure, analysis.

Examples: attribution model build, HubSpot workflow architecture, pipeline reporting.
2
Ops Associates
Documented, repeatable tasks.

Examples: post-event data upload, campaign list creation from uploaded data.
3
Marketing
Documented processes the team runs independently.

Examples: pre-event form testing, paid dashboard reporting, standard list pulls.

All via the ops request form — routed internally.

Not Sure Which Tier?

Run through these three questions first

1
How complex is this task?   Simple  ·  Medium  ·  Hard
Hard = cross-system, technical expertise needed → Tier 1. Simple = one tool, no config → likely Tier 3.
↓
2
Does this touch a system, workflow, or shared configuration?
If it involves HubSpot workflows, automations, integrations, or anything shared — come through ops, even if it looks simple. "YouTube told me how" is not enough if it touches a live system.
↓
3
Would it take me less than 1 hour and live entirely outside shared systems?
If yes — it's self-serve. If it crosses into a system or workflow at any point, use the request form regardless of how quick it seems.
Still unsure? Submit via the request form anyway — we'll triage it. You don't need to know the tier upfront.
